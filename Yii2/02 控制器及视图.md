## 02 控制器及视图
1. 命名规则

    目录：controllers  
    控制器命名：名称(首字母大写)+Controller  
    控制器方法：action+方法名(首字母大写)  
    注：文件名及类名一致。
2. 写一个简单的控制器

    controllers/TestController.php
    ```php
    namespace app\controllers;
    // vendor/yiisoft/yii2/web/Controller.php
    use yii\web\Controller;    
    class TestController extends Controller 
    {
       // 通过/index.php?r=test/index访问
       function actionIndex()
       {
           return 'test/index';
       }
       // 通过/index.php?r=test/show-user访问
       function actionShowUuser()
       {
           return 'test/show-user'
       }
    }
    ```
3. 控制器内置函数($this->function)
    
    ```php
    // 重定向
    $this->redirect(['test/index']);
    // 回到家目录
    $this->goHome();
    // 返回上层目录
    $this->goBack();
    // 刷新当前页面
    $this->refresh();
    ```
4. 视图

    目录：views
    ```php
    // 控制器内调用:views/test/index.php(包含默认头部尾部)
    return $this->render('index', ['abc'=>1234]);
    // 1. 指定视图，必须，默认当前控制器名目录
    // 2. 数据传递，可选
     
    // 同上，不含默认头部尾部
    return $this->renderPartial('index', ['abc'=>1234]);
    ```
