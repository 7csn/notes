## 01 安装配置
官网：http://www.yiiframework.com  
中文社区：http://www.yiichina.com
1. 安装composer  
    已安装，可升级：composer self-update
2. 安装yii

    ```shell
    composer global require "fxp/composer-asset-plugin:~1.1.1"
    composer create-project --prefer-dist yiisoft/yii2-app-basic yii
    ```
3. 目录结构

config          配置文件  
controllers     控制器  
models          模型  
views           视图  
web             公共文件  
runtime         缓存及日志  
vendor          核心文件
